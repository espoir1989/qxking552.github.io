Apache下mod_rpaf模块安装

mod_rpaf是apache的一个模块，目的是为了获取访客的真实IP，下面是安装方法

安装apache开发组件

yum install httpd-devel

下载插件并编译安装

wget ftp://ftp.tr.debian.org/gentoo/distfiles/mod_rpaf-0.6.tar.gz
tar xvfz mod_rpaf-0.6.tar.gz
cd mod_rpaf-0.6
apxs -i -c -n mod_rpaf-2.0.so mod_rpaf-2.0.c
chmod 755 /usr/lib64/httpd/modules/mod_rpaf-2.0.so

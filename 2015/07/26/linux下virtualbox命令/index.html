linux下virtualbox命令

  • OPS

查看当前虚拟机

VBxoManage list vms

查看当前正在运行的虚拟机

VBoxManage list runningvms

启动虚拟机

VBoxManage startvm win2

无前端图形界面方式启动虚拟机

VBoxManage startvm win2 --type headless

使用 VRDP 方式通过命令行启动虚拟机： (3389)

VBoxManage startvm win2 --type vrdp

关闭虚拟机

VBoxManage controlvm win2 poweroff
VBoxManage [-v|-version]         显示virtualbox的版本号
VBoxManage -nologo               隐藏logo

VBoxManage -convertSettingsIgnore 允许自动转换设置文件，但是不保存结果
